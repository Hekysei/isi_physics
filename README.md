## **📖 О проекте**

Данный проект представляет собой полноценное веб\-приложение, созданное для визуализации и математического расчета различных физических процессов. Цель проекта — объединить знания из области программирования и физики для создания удобного инструмента, который поможет студентам и школьникам наглядно изучать законы природы.

При разработке мы сделали упор на качество кода, безопасность и современную архитектуру.

## **🛠 Стек технологий и Архитектура**

Проект построен на современном стеке Java с использованием передовых практик разработки:

### **Технологии**

* **Язык:** Java 17 (или выше)  
* **Фреймворк:** Spring Boot (ускоренная настройка и развертывание)  
* **Безопасность:** Spring Security (аутентификация пользователей, защита маршрутов, разделение ролей: Студент / Преподаватель)  
* **Сборка:** Maven (управление зависимостями и жизненным циклом проекта)

### **Архитектура и Паттерны**

* **MVC (Model-View-Controller):** Строгое разделение логики работы с данными, бизнес-логики и пользовательского интерфейса.  
* **SOLID:** Архитектура приложения спроектирована с соблюдением пяти базовых принципов объектно-ориентированного программирования, что делает код масштабируемым, легко тестируемым и поддерживаемым.

## **⚙️ Запуск проекта (Локально)**

Для запуска проекта на вашем локальном компьютере потребуется установленные **Java Development Kit (JDK)** и **Maven**.

### **1\. Переход в корневую папку проекта**

```powershell
cd "c:\вашадиректория\isi_physics-master"
```

### **2\. Сборка проекта**

Выполните сборку проекта с помощью Maven:

```powershell
mvn clean package
```

### **3\. Запуск приложения**

Запустите приложение из директории проекта:

```powershell
mvn spring-boot:run
```

### **4\. Альтернативный запуск**

Если вы уже собрали JAR, можно запустить его напрямую:

```powershell
java -jar target\isib-1.0.0.jar
```

### **5\. Доступ к приложению**

После успешного запуска приложение будет доступно в браузере по адресу:

```text
http://localhost:8080/
```

### **6\. Дополнительная документация**

Подробный мануал запуска находится в файле `RUNNING.md`.
